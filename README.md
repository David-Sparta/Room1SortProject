## An array sorting app.
The user inputs an array size and type of sort to be used. <br/>
The app returns the now sorted array, sorted to user specification.  <br/>
Types of algorithm to choose from are: <br/>
BubbleSort <br/>
MergeSort <br/>
LibrarySort <br/>
